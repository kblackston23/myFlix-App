myFlix


This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Response
Get a list of movies /Movies GET Returns a JSON list of movies.
Get data about the movie /movies/:title GET Returns a JSON data about a specific movie.
Return data about a genre by name/title. /genres/:genre GET Returns a JSON data with movies of that ganre.
Get data about the director /directors/:directorName GET Returns a JSON data about auther year.
Add new user /users/:newUser POST Will add new user to JSON object.
Allow users to update their user information /users/:Username PUT Returns the data fields that was updated by the user.
Delete the user /users/:deleteUser DELETE Return a text confirming that the user was deleted.
Add new movie to list of favorite /favorite/:movieName POST Returns a text confirming which movie was added successfully.
Delete movie from list of favorite /favorite/:deleteMovie DELETE Returns a text confirming which movie was removed successfully.
